## Getting Started 
To get a local copy up and running follow these simple example steps.

### Prerequisites

Make sure Node.js is running on your local machine

### Setup

~~~bash
$ git clone https://github.com/Godswilly/react-calculator.git
$ cd react-calculator
~~~

Install modules:

```
yarn install
```

### Usage

Start server with:

```
    yarn start or npm start
```
