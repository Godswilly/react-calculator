# React Calculator App
This project is a basic react calculator app with logic and no functionality.

## Built With
- CSS
- JavaScript
- React

## Live Demo
[Live Demo](https://angry-golick-6671e1.netlify.app/)

## Getting Started 
To get a local copy up and running, follow these simple example steps.

### Prerequisites

Make sure you have Node.js  running on your local machine.

### Setup

~~~bash
$ git clone https://github.com/Godswilly/react-calculator.git
$ cd react-calculator
~~~

Install modules:

```
yarn install
```

### Usage

Start server with:

```
    yarn start or npm start
```
